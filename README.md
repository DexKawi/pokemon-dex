# Pokemon-dex

> A simple Next.js application to view Pokemon details using the PokeAPI to improve my understanding about ReactJS and NextJS essentials.

## Features

- View detailed Pokemon information
- Display Pokemon species data
- Clean and simple interface

## Tech Stack

- Next.js (full-stack)
- PokeAPI (pokemon & pokemon-species endpoints)

## API Endpoints Used

- `https://pokeapi.co/api/v2/pokemon/{id or name}` - Pokemon data
- `https://pokeapi.co/api/v2/pokemon-species/{id or name}` - Species data
